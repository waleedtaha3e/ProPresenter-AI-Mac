# Baseline Add-in for Mac

PowerPoint add-in (Office.js + TypeScript) replicating baseline Efficient Elements features with dynamic settings.

## Features
- Alignment & positioning tools
- Configurable grid sizes and standard dimensions
- Dynamic settings panel
- Branding presets

## Setup
1. Install Node.js (LTS).
2. Clone repo or download ZIP.
3. Install dependencies:
   ```bash
   npm install
   npm start
